= singleton-rs: Helloworld Using JAX-RS (Java API for RESTful Web Services)

[abstract]
The `singleton-rs` quickstart demonstrates a simple Hello World application, bundled and deployed as a WAR, that uses JAX-RS to say Hello.


== What is it?

The `singleton-rs` quickstart demonstrates the use of CDI and JAX-RS in Jakarta EE.

== Access the Application

The application is deployed to http://localhost:8080/{artifactId}/.

* The _XML_ content can be viewed by accessing the following URL: http://localhost:8080/{artifactId}/rest/xml

* The _JSON_ content can be viewed by accessing this URL: http://localhost:8080/{artifactId}/rest/json
